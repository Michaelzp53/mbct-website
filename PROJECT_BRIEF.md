# MBCT 迈创兄弟 官网搭建简报

## 1. 项目概述
- **项目名称**: MBCT Corporate Website (中英双语)
- **核心定位**: 酒店行业AI数字化与私域流量增长专家
- **目标**: 建立国际化专业形象，触达全球酒店行业客户
- **网址**: marvelbros.com

## 2. 设计风格
- **主色调**: 深空蓝 #0f172a + 星光金 #f59e0b
- **风格**: 现代科技感、酒店行业专业、深色主题
- **字体**: Inter (英文) + Noto Sans SC (中文)
- **参考**: 旧站结构 + Cloudbeds/Mews/Oaky 等国际酒店科技公司风格

## 3. 技术栈
- **框架**: Next.js 15 (App Router)
- **样式**: Tailwind CSS v4
- **组件**: shadcn/ui
- **部署**: Vercel
- **语言**: TypeScript

## 4. 页面架构 (MVP)

### 核心页面 (双语)
1. **首页** (/) - Hero + 8大服务概览 + 核心优势 + 案例精选 + CTA
2. **关于我们** (/about) - 公司介绍、团队、资质
3. **服务内容** (/services) - 8大支柱详解
4. **案例展示** (/cases) - 成功案例卡片
5. **联系我们** (/contact) - 咨询表单 + 联系方式

### URL结构
- 中文: /zh/*
- 英文: /en/*
- 默认跳转: / → /zh/

### 中英文切换
- URL路由方式 (/zh/ 和 /en/)
- 导航栏语言切换按钮

## 5. 内容 (先做骨架，内容后续填充)

### 首页模块
- Hero: 全屏视觉 + 主标语 + CTA按钮
- 8大支柱概览: 图标 + 标题 + 简述
- 核心优势: 3-4个差异化亮点
- 案例精选: 6个标杆案例卡片
- CTA区: 立即咨询/预约演示

### 8大服务支柱
1. 酒店投资可行性研究
2. 酒店品牌创建与定位全案
3. 酒店运营诊断与增长服务
4. 酒店AI管理系统（SaaS）
5. 酒店自媒体及私域流量运营
6. 落地陪跑与效果对赌
7. 酒店轻量管理顾问
8. 酒店融资顾问

## 6. 设计规范

### 色彩系统
- 深空蓝(主色): #0f172a
- 星光金(品牌色): #f59e0b
- 电光蓝(辅助): #3b82f6
- 增长绿(成功): #22c55e
- 极夜黑(深色背景): #111827
- 月光白(浅色背景): #f8fafc

### 间距系统
- 基准: 16px
- 页面最大宽度: 1400px
- 桌面边距: 80px
- 移动边距: 16px

### 圆角与阴影
- 按钮: 10px圆角
- 卡片: 16px圆角
- 悬停: 上移2px + 阴影加深

### 导航栏
- 高度: 70px, sticky定位
- 背景: 玻璃拟态效果
- Logo: 左侧
- 导航: 居中
- CTA按钮: 右侧
- 语言切换: 最右侧
